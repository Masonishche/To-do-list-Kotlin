# Простий список справ (To-Do List) 

Це тестове завдання на позицію Android Kotlin (Jetpack Compose) Trainee. Додаток дозволяє користувачам створювати список завдань та керувати ним.

##  Функціонал
* **Додавання справ:** Поле вводу та кнопка для створення нових завдань.
* **Список:** Відображення списку справ з прокруткою (LazyColumn).
* **Видалення:** Можливість видаляти справи зі списку.
* **Реактивність:** Інтерфейс автоматично оновлюється при зміні даних завдяки Jetpack Compose.
* **In-Memory зберігання:** Дані зберігаються в оперативній пам'яті під час роботи додатка (згідно з ТЗ).

## Технології
* **Мова:** Kotlin
* **UI:** Jetpack Compose (Material Design 3)
* **IDE:** Android Studio
* **Збірка:** Gradle

## Як запустити
1.  Клонуйте цей репозиторій.
2.  Відкрийте проект в **Android Studio**.
3.  Дочекайтеся синхронізації Gradle.
4.  Запустіть додаток на емуляторі або підключеному Android-смартфоні.
